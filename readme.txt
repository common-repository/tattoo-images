== Description ==

[Tattoo Images] is a WordPress plugin that let's you pick pictures from [Tattmight.com](http://tattmight.com/) and insert them with just a click anywhere on your blog. The images are safe to use, linking back to the source is required.

== Installation ==

1. Install Tattoo Images either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it. You're ready to go!

== Frequently Asked Questions ==

= Are there any restrictions or requirements for using these images? =

You are free to adapt and use the images. Please leave the original author or source link back to gallery.
